# YOLO-ICP pipeline for object 6D pose estimation

The file structure should be in the following

    * cpp
    * Imo_models
    * models
    * python
    * train_pbt

The train_pbt is not uploaded because there are too many files. The train_pbt can be downloaded from the BOP website using the following link https://huggingface.co/datasets/bop-benchmark/lm/resolve/main/lm_train_pbr.zip. Extract all the files and place them into the working directory.
